# Model Training Assets

This directory holds local, large, non-git assets:

- datasets
- checkpoints
- exported inference bundles

## Recommended layout

- `datasets/llm-recommender/`
- `datasets/image-processor/`
- `checkpoints/llm-recommender/`
- `checkpoints/image-processor/`
- `outputs/`

## Rule

Keep real weights, raw dataset dumps, and training artifacts here.

Do not commit them to git.
